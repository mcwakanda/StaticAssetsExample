This page illustrates techniques for storing your static assets in the Wakanda Datastore
1. Upload to server folder
2. Upload to datastore image attribute
Note: This technique requires that a saved entity exist on the server prior to upload. This is the reason widgets disable at certain times.
3. Build form, save to folder, associate to attribute
Note: This example does not require a saved entity to be the current selection. It builds a form submission dynamically.
